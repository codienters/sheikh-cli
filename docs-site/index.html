AI agents your unfair advantage - powerful AI capabilities in your terminal. Build agents and workflows with our API. Turn hours-long workflows into a single command.

Get Started View on GitHub

Quick Start

Install

Install Sheikh-CLI globally using npm

npm install -g @codienters/sheikh-cli
Configure

Set up your API key and configuration

sheikh config --init
Start Chatting

Begin your AI-powered development journey

sheikh chat --agentic

Key Features

Agentic Search

Understand your entire codebase without manual context selection. Search million-line codebases instantly with semantic understanding.

Multi-File Coordination

Make coordinated changes across multiple files. The system analyzes all related files and plans coordinated changes.

Visual Diffs

See visual diffs and approve changes before they're applied. Never modifies files without explicit approval.

Workflow Generation

Turn natural language descriptions into executable workflows. Generate complex workflows from simple descriptions.

Sheikh-CLI ডকুমেন্টেশন

AI এজেন্ট আপনার অন্যায্য সুবিধা - টার্মিনালে শক্তিশালী AI ক্ষমতা। আমাদের API দিয়ে এজেন্ট এবং ওয়ার্কফ্লো তৈরি করুন। ঘন্টার পর ঘন্টার ওয়ার্কফ্লোকে একটি কমান্ডে পরিণত করুন।

শুরু করুন GitHub এ দেখুন

দ্রুত শুরু

ইনস্টল করুন

npm ব্যবহার করে Sheikh-CLI গ্লোবালি ইনস্টল করুন

npm install -g @codienters/sheikh-cli
কনফিগার করুন

আপনার API কী এবং কনফিগারেশন সেট আপ করুন

sheikh config --init
চ্যাট শুরু করুন

আপনার AI-চালিত ডেভেলপমেন্ট যাত্রা শুরু করুন

sheikh chat --agentic

মূল বৈশিষ্ট্য

এজেন্টিক সার্চ

ম্যানুয়াল কনটেক্সট নির্বাচন ছাড়াই আপনার পুরো কোডবেস বুঝুন। সেমান্টিক বোঝাপড়া দিয়ে মিলিয়ন-লাইন কোডবেস তাত্ক্ষণিকভাবে সার্চ করুন।

মাল্টি-ফাইল সমন্বয়

একাধিক ফাইলে সমন্বিত পরিবর্তন করুন। সিস্টেম সব সম্পর্কিত ফাইল বিশ্লেষণ করে এবং সমন্বিত পরিবর্তনের পরিকল্পনা করে।

ভিজুয়াল ডিফ

পরিবর্তন প্রয়োগের আগে ভিজুয়াল ডিফ দেখুন এবং অনুমোদন দিন। স্পষ্ট অনুমোদন ছাড়া কখনই ফাইল পরিবর্তন করে না।

ওয়ার্কফ্লো জেনারেশন

প্রাকৃতিক ভাষার বর্ণনা থেকে এক্সিকিউটেবল ওয়ার্কফ্লো তৈরি করুন। সহজ বর্ণনা থেকে জটিল ওয়ার্কফ্লো জেনারেট করুন।
